# CREDITARIA FRONTEND

Características solicitadas.

1. Formulario de entreada y tabla de resultados

2. Manejo de estado inteligente

    - Si el usuario cierra la pestaña y vuelva abrirla, los campos del formulario
    dben recordar los últimos valores.

    - Sin embargo, si el usuario cambia el valor del **Monto** la tabla de resultados
    debe desaparecer obligando al usuario hacer clic en el botón **Calcular**
    para ver los datos actualizados.
